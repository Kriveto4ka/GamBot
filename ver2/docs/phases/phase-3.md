# Phase 3: Game Mechanics

## Цели этапа
Реализация основной игровой логики: начисление опыта (XP), система уровней, проверка дедлайнов и нанесение урона.

## Требования

### 1. Система опыта и уровней
- **Формула уровня:** `XP для уровня N = 100 * N`
- **Повышение уровня:**
    - XP сбрасывается до 0 (излишек переносится)
    - Max HP += 10
    - HP восстанавливается до 100%
- **Начисление XP:**
    - Лёгкая: +10 XP
    - Средняя: +25 XP
    - Сложная: +50 XP
    - Эпическая: +100 XP

### 2. Механика урона
- **Урон за просрочку:**
    - Лёгкая: -5 HP
    - Средняя: -15 HP
    - Сложная: -30 HP
    - Эпическая: -50 HP
- **Смерть:**
    - При HP <= 0
    - Сброс уровня до 1
    - Сброс XP до 0
    - Сброс HP/MaxHP до 100
    - Удаление всех активных задач

### 3. Планировщик (Scheduler)
- Проверка дедлайнов каждые 5 минут
- При просрочке:
    - Нанести урон
    - Пометить задачу как просроченную (failed)
    - Отправить уведомление пользователю
